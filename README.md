# AP District-Wise Water Body Analysis Using Data Mining

## Overview
This project analyzes Andhra Pradesh water body census data using Data Mining techniques to identify underutilized, critical, and high-value water bodies.

## Objectives
- Analyze water body distribution across AP districts
- Identify vulnerability zones
- Discover hidden patterns in water resource utilization
- Support data-driven decision making

## Techniques Used
- Exploratory Data Analysis (EDA)
- K-Means Clustering
- Decision Tree (CART)
- Apriori Association Rule Mining

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Key Findings
- 75.5% of water bodies are underutilized
- 8.7% are critical and require intervention
- 15.8% are high-value water bodies
